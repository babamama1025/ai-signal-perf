# AI 號誌績效比較分析工具

以 Streamlit 建立的互動式網頁應用程式，用於比較 AI 號誌（事後）與定時時制（事前）之交通績效差異。

## 功能概覽

- **互動式日期分配**：自動從 Excel 測試日工作表讀取事前/事後分類，可於介面手動調整勾選
- **多時段分析**：可同時選取多個時段（例如早峰、下午峰）一次執行比較
- **四項績效指標**：
  - 總停等延滯（秒）
  - 通過量（輛）
  - 平均停等延滯（秒/輛，以 mean(總延滯)/mean(通過量) 計算，避免加權偏差）
  - 旅行時間廊道（秒）
- **多層級比較**：系統總量 → 各路口聚合 → 各來向明細
- **Plotly 視覺化**：改善率概覽圖、事前 vs 事後群組長條圖、旅行時間廊道比較圖
- **Excel 報告匯出**：含分析說明、總表、各時段明細工作表，可選加入原始資料

## 專案結構

```
.
├── performance_comparison_app.py   # Streamlit 主程式
├── data_loader.py                  # CSV/Excel 資料讀取與欄位結構自動偵測
├── comparison_logic.py             # 統計計算與分析文字產生
├── chart_builder.py                # Plotly 圖表工廠
├── export_builder.py               # openpyxl Excel 匯出
├── performance_summary.csv         # 績效原始資料（UTF-8）
├── =*績效*.xlsx                    # 測試日分類來源（選用）
└── run.bat                         # Windows 一鍵啟動腳本
```

## 資料格式

`performance_summary.csv` 需包含以下固定欄位，後續欄位由程式自動偵測：

| 欄位 | 說明 | 範例 |
|------|------|------|
| 日期 | `YYYY/MM/DD` 或 `YYYY-MM-DD` | `2025/04/01` |
| 時段 | `HH:MM~HH:MM` | `07:00~09:00` |
| 指標 | 量測指標名稱 | `總停等延滯` |
| 系統 | 系統總量 | 數值欄 |
| `2 青昇` | 路口聚合欄（數字開頭） | 數值欄 |
| `A`、`B.1` | 來向欄（單一大寫字母，可加 `.數字` 後綴） | 數值欄 |
| `廊道->終點` 或 `路徑名稱` | 旅行時間廊道欄（欄名含「->」或「路徑」） | 數值欄 |

`=*績效*.xlsx`（選用）需包含名為「測試日」的工作表，第 3 列起每列第 2 欄為日期，第 4～11 欄以布林值標記 AI 測試日。

## 環境需求

- Python 3.10+
- 套件：`streamlit`、`pandas`、`plotly`、`openpyxl`

安裝：

```bash
pip install streamlit pandas plotly openpyxl
```

## 啟動方式

**Windows（建議）：**

直接雙擊 `run.bat`，或於命令提示字元執行：

```bat
run.bat
```

**命令列：**

```bash
python -m streamlit run performance_comparison_app.py
```

啟動後開啟瀏覽器至 `http://localhost:8501`。

## 操作步驟

1. 側邊欄的日期分配表格會依「測試日」工作表自動預填事前（FIX）/事後（AI）
2. 確認或手動勾選每日的事前/事後歸屬（點「重置」可還原自動分類）
3. 選擇要分析的時段（可多選）
4. 視需要勾選「包含旅行時間」、「顯示各路口各方向明細」
5. 點擊「執行分析」
6. 查看圖表與分析摘要後，可點「產生 Excel 報告」下載完整報告

## 計算原則

- **總停等延滯、通過量**：取選定日期的欄位算術平均值
- **平均停等延滯**：`mean(總停等延滯) / mean(通過量)`，不使用 CSV 中已存在的平均停等延滯列，以避免「平均的平均」造成的加權偏差
- **改善率**：
  - 越小越好（延滯、旅行時間）：`(事前 − 事後) / 事前`
  - 越大越好（通過量）：`(事後 − 事前) / 事前`

## 模組說明

| 模組 | 職責 |
|------|------|
| `data_loader.py` | 讀取 CSV、自動偵測路口/來向/旅行時間欄位結構、從 Excel 讀取測試日分類 |
| `comparison_logic.py` | 依時段與日期集合計算各指標的事前/事後均值與改善率，產生文字摘要 |
| `chart_builder.py` | 以 Plotly 建立概覽圖、指標長條圖、旅行時間廊道比較圖 |
| `export_builder.py` | 以 openpyxl 輸出含樣式的 Excel 報告（分析說明、總表、各時段、原始資料） |
